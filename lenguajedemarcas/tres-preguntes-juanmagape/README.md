# PhoenixTech — Checkpoint de coneixement

Aquest repositori conté l’eina que utilitzarem per respondre les preguntes del checkpoint del projecte **PhoenixTech**.

## Objectiu

Cada alumne haurà de respondre **3 preguntes** sobre:

- el projecte PhoenixTech
- JSON
- Git / GitHub

Les preguntes es generen **aleatòriament**.

---

## Com fer l'activitat

## 1. Obre l'eina

Obre el fitxer `tres-preguntes.html` amb el navegador.

---

## 2. Selecciona el teu nom

Selecciona el teu nom al desplegable.

---

## 3. Respon les preguntes

Respon **les 3 preguntes assignades**.

Totes les respostes són **obligatòries**.

---

## 4. Descarrega el fitxer JSON

Quan hagis acabat, prem el botó:

```bash
Descarrega respostes JSON
```

Això generarà un fitxer **`.json`** amb les teves respostes.

---

## 5. Guarda el fitxer al repositori

- Mou el fitxer descarregat a la carpeta `respostes/` i

- assegura't que el nom és `answers.json`

L'estructura final del repositori ha de quedar així:

```bash
repo
├── tres-preguntes.html
├── README.md
├── fitxers
│   ├── control.js
│   ├── estils.css
│   └── tres-preguntes.html
└── respostes
    └── answers.json
```

---

## 6. Fes commit i push

Després d'afegir el fitxer:

```bash
git add .
git commit -m "Respostes tres preguntes"
git push
```

---

## Important

- Les preguntes són **aleatòries**.
- No es poden canviar recarregant la pàgina.
- Respon amb **les teves pròpies paraules**.
