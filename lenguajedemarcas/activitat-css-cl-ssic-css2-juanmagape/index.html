Windows té una caiguda de mercat degut a l'obligació d'actualitzar a Windows 11

Windows 11, Cuota de mercat, Actualització

Microsoft ha experimentat una disminució en la seva quota de mercat després d’imposar l’actualització obligatòria a Windows 11. Molts usuaris han mostrat descontentament a causa dels requisits tècnics més estrictes i problemes de compatibilitat amb equips antics.
